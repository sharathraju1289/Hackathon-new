# Hackathon-project

Real estate web app built with React + Vite.

## Features
- Property gallery with 100+ house images
- Fullscreen image gallery
- Chatbot integration
- Wishlist &amp; Schedules
- Responsive design with Framer Motion animations

## Quick Start
```bash
npm install
npm run dev
```

## Scripts
- `npm run download-images`: Download house images

Built for hackathon.
